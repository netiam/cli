> The netiam CLI tool

![netiam-cli](docs/videos/netiam-cli.gif)

## Install

```bash
npm i -g netiam-cli
```

## Usage

### Create new project

```bash
netiam-cli create myapi
```

### Initialize in current directory

```bash
netiam-cli init
```

### Create new resource

```bash
netiam-cli generate resource users
```
